# Lab 5 - EKF SLAM

In this lab you will implement EKF SLAM and data association for both simulated and real-world datasets.

## Code

There are various options that can be set when executing run.py
Run ```./run.py -h``` to see all the options listed.

For example, running the following commands will run the simulator dataset with known data association for 100 times steps.
```
cd code
./run.py -t sim -d known -n 100
```
